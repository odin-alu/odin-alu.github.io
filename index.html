About IP 2

Web Development Assigment

IP 2 is a collaborative web development assignment at the end of our third year at ALU Mauritius. The project is carried in a group of 4, where it covers requirements to design, collaborative development to project management.


Our project is Odin, a website that makes use of different APIs presented in dynamic and interactive visualisations. Furthermore, a tutorial on GeoJSON is available for absolute beginners.

Download Requirements

Requirements

Design Process

Brainstorming about the UI of the website and creating the sitemap. Creating the wireframes, mockups, and iterating.

Prototyping

Designing and developing both static and interactive prototypes using Figma.

Research

Assess similar websites to gather insights about different user experiences, to then designing our own.

Testing

From unit testing to integration testing post-development of individual charts. Regression testing post-integration.

Libraries

Evaluating open-source libraries and choosing them before starting the development. For e.g, Bootstrap Grid Layout.

Collaboration

Collaborative development of the website through GitHub, and managing the project via Trello.
